Powder Library

320+ propellant powders — burn rate rankings, type, and applications. Sorted fastest to slowest.

This reference covers 320+ reloading powders across all major brands — Hodgdon, IMR, Alliant, Vihtavuori, Accurate, Ramshot, Winchester, Norma, ADI, Shooters World, and more. Each entry includes burn rate rank (1 = fastest, 100 = slowest), powder type (Ball, Extruded, Disc, Flake), best application (Pistol, Rifle, Shotgun), temperature sensitivity, and recommended calibers.

Burn rate position is relative across the spectrum — use this to compare powders within a category and find substitutes. Always consult a current manufacturer reloading manual for actual charge weights and safe load development. Never exceed published maximum loads.
